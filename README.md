## 莲池学堂
欢迎来莲池学堂

## 莲池学堂是什么？
莲池学堂是一个致力于传播中国传统文化的免费社区

让普通人也能享受到专业传统文化和传统中医教育的社区。

## 莲池学堂的特点
莲池学堂是基于github最受欢迎的学习程序freecodecamp修改的。

因此其教学形式已经得到实际验证

## 为什么要创建莲池学堂？
中国传统文化一直没有一个系统的教学课程。
